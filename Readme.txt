Controls for program:

================= Assignment 4: Robot Hunter =================
Keyboard Controls:
  w   : Wireframe mode (entire scene)
  s   : Solid mode (entire scene)
  c   : Toggle collider display (robots + obstacles)
  a   : Toggle world axes at origin
  b   : Toggle bullet speed (slow, fast, very fast)
  m   : Toggle robot motion (walk / jump disruption)
  l/L : Toggle lighting type (directional / point)
  o/O : Toggle imported 3D model and shelf obstacles on/off
  Space : Fire bullet
  ESC   : Pause menu (NEW GAME / RESUME / EXIT)
  F1    : Toggle fullscreen / windowed
  F2    : Swap main FPV / ESV view
  F3    : Toggle game sound on/off
  F4    : Toggle shading (smooth Gouraud / flat)

Camera Controls:
  Arrow Keys (FPV main): move and rotate FPV camera
  Arcball (ESV main only):
    Left-drag  : rotate camera
    Right-drag : zoom in/out

Mouse Controls:
  ESC menu: Use arrow keys + Enter/Space or left-click buttons

Game Info:
  Score: +10 for each robot hit, -2 for each missed shot.
  Time limit: 30 seconds; eliminate all robots to complete the mission.
  All keyboard controls are disabled once the mission ends.
