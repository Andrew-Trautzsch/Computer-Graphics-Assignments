Controls for program:


================= Robot Controls =================
Keyboard Controls:
  p : Vertex-only model
  w : Wireframe model
  s : Solid model
  c : Clear screen toggle
  a : Toggle axes on/off
  d : Toggle dancing animation on/off
  m : Toggle music on/off
  h : Play hello audio
  j : Play dance audio
  k : Play goodbye audio
  q : Quit program

Mouse Controls:
  Right click : Open popup menu
  Left click  : Toggle between B&W and Color
  Middle scroll : Zoom in/out

Menu Options:
  - Rotate robot (angles 30-360) along X, Y, Z axes
  - Projection: Orthographic (default) or Perspective
  - Change body part colors (Head, Torso, Arms, Legs)
